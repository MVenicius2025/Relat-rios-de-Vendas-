# 📊 Relatório de Vendas Elegante com Power BI

Este projeto foi desenvolvido como parte do desafio da DIO **"Criando um Relatório de Vendas Elegante com Power BI"**.  
O objetivo é construir um dashboard interativo e visualmente atraente utilizando a amostra **Financials** do Power BI.

---

## 🚀 Funcionalidades
- [Botões de navegação](ca://s?q=Bot%C3%B5es_de_navega%C3%A7%C3%A3o_no_Power_BI) para facilitar a interação entre páginas.
- [Segmentadores de dados](ca://s?q=Segmentadores_de_dados_no_Power_BI) com imagens associadas.
- [Indicadores e bookmarks](ca://s?q=Indicadores_e_bookmarks_no_Power_BI) para alternar entre diferentes visuais.
- [Publicação no Power BI Service](ca://s?q=Publicar_relatorio_no_Power_BI_Service) para compartilhamento online.

---

## 📂 Estrutura do Repositório
- `Relatorio.pbix` → Arquivo principal do Power BI.  
- `Dados/` → Dataset sample financials.  
- `Imagens/` → Capturas de tela do relatório.  
- `README.md` → Documentação do projeto.

---

## 🖼️ Layout do Relatório
- Página 1: KPIs principais, gráficos de vendas e botões de navegação.  
- Página 2: Detalhamento por país, ano e categoria de produto.
